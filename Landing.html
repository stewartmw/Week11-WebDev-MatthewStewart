Summary: Weather vs. Latitude


The purpose of this project was to analyze various weather phenomena vs. latitude. In particular, we wanted to analyze the following:

  • Temperature
  • Humidity
  • Wind Speed
  • Cloud Cover

We were first provided a dataset containing a list of over 500 randomly selected cities. This datset contained multiple fields of information, including the name of the city and the four weather phenomena mentioned above. The goal was to visualize this data and examine whether any patterns could be detected as location approached the equator (i.e., as location approached latitude of 0). Note that this data was gathered on June 13, 2018.

Contained within this site, you will find links to an overall comparisons page, in addition to links to the individual visualizations mentioned above. Finally, you will be able to view the raw data for yourself using the Data link.
